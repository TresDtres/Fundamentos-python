1. Variables y `print()`

Una variable es un nombre para guardar un valor. La función print() muestra valores en pantalla.

2. Tipos de Datos

Los datos pueden ser de tipo str (texto), int (entero), float (decimal) o bool (True/False).

3. Operadores

Permiten hacer cálculos (+, -, *, /, //, %) y comparaciones (==, >).

4. Listas

Son colecciones ordenadas de datos que se crean con []. Se accede por su índice (posición) empezando en 0.

5. Condicionales (`if`, `else`)

Permiten que el código tome decisiones y siga un camino u otro.

6. Bucles (`for` y `while`)

Sirven para repetir tareas sobre una lista (for) o mientras se cumpla una condición (while).

7. Funciones (`def`)

Permiten crear bloques de código reutilizables para no repetirnos.

8. Diccionarios

Son colecciones para guardar datos con pares clave: valor. Se crean con {}.

9. Combinar Listas y Diccionarios

La estructura más común: una lista donde cada elemento es un diccionario.
